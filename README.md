# EBOOK-MANAGER-OCR

A secure, offline-first desktop app to digitize, manage, and search your handwritten notes.

It solves the "privacy vs. convenience" problem by performing all OCR and storage 100% offline. Your data never leaves your machine.

Key Tech: Java 21, Swing, Tesseract OCR, JSON/File-System Storage

Prerequisites: Java JDK 21, Apache Maven, and Tesseract OCR Engine (must be added to system PATH).
